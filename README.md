# rest_and_chill_app

ChillApp
This application is aimed at testing the user for burnout and fatigue using the ML model, and then in a game form giving him advice on how to help himself using in-game currency

Using Spring, Docker, Kafka, Camunda BPMN, PostgreSQL, Pandas, Matplotlib 

Core API — entry point for the client.
Camunda Service — business logic.
Economy Service — coin and reward logic.
ML Service — questionnaire evaluation.
